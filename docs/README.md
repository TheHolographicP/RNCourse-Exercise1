# Course Goals App

A simple Expo React Native app for managing course goals. Users can add goals via a text input and delete them by tapping.

![App UI](./assets/AppUI.png)

## Features
- Enter a goal in the "Course Goal" text input.
- Add the goal to a list by pressing the add button.
- Tap on any goal in the list to delete it.

## How to Run
1. Ensure you have Expo CLI installed (`npm install -g @expo/cli`).
2. Navigate to the project root
3. Install dependencies: `npm install`.
4. Start the app: `expo start`.
5. Scan the QR code with the Expo Go app on your device or run in an emulator.
